# 甲状腺超声图像分割模型对比实验报告

## 1. 实验目的

本实验旨在对比不同深度学习模型在同一套参数下对甲状腺超声 TN3K 数据集的分割效果，评估各模型的性能表现，为甲状腺结节分割任务选择最优模型。

## 2. 实验数据集

- **数据集名称**：TN3K (Thyroid Nodule Ultrasound Dataset)
- **数据类型**：甲状腺超声图像
- **数据格式**：JPG 格式
- **数据预处理**：
  - 图像大小调整为 448×384
  - ZScore 归一化
  - 二值化处理（阈值 128）
- **数据分割**：80% 训练集，20% 验证集

## 3. 实验设置

### 3.1 模型选择

本实验对比了以下 8 种分割模型：

1. UNet
2. Attention UNet
3. UNet++
4. Inception UNet
5. TND SEG Net
6. MK UNet
7. TransUNet
8. nnUNet (PlainUNet)

### 3.2 训练参数

所有模型使用相同的训练参数：

- **批量大小**：16
- **初始学习率**：0.0001
- **训练轮次**：50
- **学习率调度器**：余弦退火
- **损失函数**：DICE + BCE 组合损失
- **优化器**：Adam
- **随机种子**：42
- **混合精度训练**：启用

### 3.3 评估指标

- **DICE 系数**：主要评估指标
- **IOU (交并比)**
- **召回率 (Recall)**
- **精确率 (Precision)**

## 5. 分析与讨论

### 5.1 模型性能分析

1. **nnUNet** 和 **MKUNet** 表现最佳，DICE 系数均超过 0.81，说明这两种模型在甲状腺结节分割任务中具有优异的性能。

2. **UNetPlusPlus** 和 **TNDSEGNet** 表现次之，DICE 系数约为 0.748，属于中等偏上水平。

3. **InceptionUNet**、**UNet**、**AttentionUNet** 和 **TransUNet** 表现相对较差，DICE 系数在 0.66-0.71 之间。

### 5.2 训练效率分析

- **最快的模型**：UNetPlusPlus 和 TNDSEGNet，平均每轮训练时间约 11.6 秒
- **最慢的模型**：AttentionUNet，平均每轮训练时间约 38.0 秒
- **其他模型**：训练时间在 13-24 秒之间

### 5.3 模型复杂度与性能关系

- **复杂度较高的模型**：AttentionUNet、MKUNet、UNet
- **复杂度较低的模型**：UNetPlusPlus、TNDSEGNet、InceptionUNet
- **性能与复杂度关系**：模型性能并不完全与复杂度成正比，nnUNet 和 MKUNet 虽然复杂度较高，但表现最佳；而 UNetPlusPlus 和 TNDSEGNet 复杂度较低但表现也不错。

## 6. 结论

1. **最优模型**：nnUNet 在本次实验中表现最佳，DICE 系数达到 0.8177，是甲状腺结节分割任务的最佳选择。

2. **次优模型**：MKUNet 表现接近 nnUNet，DICE 系数达到 0.8140，也是一个很好的选择。

3. **效率与性能平衡**：UNetPlusPlus 和 TNDSEGNet 在保证一定性能的同时，具有较快的训练速度，适合资源受限的场景。

4. **实验价值**：本实验为甲状腺结节分割任务提供了模型选择的参考，同时验证了统一参数设置下不同模型的表现差异。

## 7. 未来工作

1. **模型融合**：考虑将多个模型进行融合，进一步提高分割性能。

2. **数据增强**：增加更多的数据增强策略，提高模型的泛化能力。

3. **超参数优化**：针对每个模型进行单独的超参数优化，进一步提高性能。

4. **临床验证**：将最佳模型应用于临床实践，验证其实际应用价值。
